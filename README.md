## Français

### Description

EducTool est un logiciel de gestion d'établissement scolaire complet et moderne. Conçu pour simplifier l'administration des écoles, collèges et lycées, il offre une solution tout-en-un pour gérer efficacement les élèves, les enseignants, les emplois du temps, les notes et bien plus encore.

### Technologies

- **Tauri** - Framework permettant de créer des applications de bureau légères et performantes
- **React** - Bibliothèque JavaScript pour une interface utilisateur réactive et moderne.

### Fonctionnalités principales

- Gestion des élèves et des dossiers scolaires
- Gestion du personnel enseignant et administratif
- Planification des emplois du temps
- Suivi des notes et bulletins scolaires
- Gestion des absences et retards
- Fonctionnement hors ligne avec synchronisation des données

## English

### Description

EducTool is a comprehensive and modern school management software. Designed to simplify the administration of primary schools, middle schools, and high schools, it offers an all-in-one solution to efficiently manage students, teachers, schedules, grades, and much more.

### Technologies

- **Tauri** - Framework for building lightweight and high-performance desktop applications
- **React** - JavaScript library for a reactive and modern user interface 

### Main Features

- Student and academic records management
- Teaching and administrative staff management
- Schedule planning
- Grade tracking and report cards
- Absence and tardiness management
- Offline functionality with data synchronization
